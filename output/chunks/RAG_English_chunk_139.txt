Code generation aims to convert Nat- ural Language (NL) descriptions into code implementations. Query-based RAG is a common method for code gen- eration. It builds prompts for transformer-based generative models with retrieved information, including similar exam- ples [40], [152], [218]–[221], relevant API details [88], [222], documentations [42], imports [223], and global func- tions [224]. SKCODER [225] retrieves relevant code snip- pets to produce sketch template for final code generation.
